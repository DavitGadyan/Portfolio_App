ABOUT ME

It’s David, Professional Data Scientist. I have various skills and experience in Data Science field. I will be glad to contribute to your projects in mutually beneficial way. If you have business idea and check to validate it with MVP or you have established services I can embark and make the journey more successful.

My skills include machine learning, deep learning, computer vision, full stack development, web scraping, data engineering, web development and data visualizations. My main programming languages are Python, R, Javascript and SQL. I can create interactive web applications and dashboards using Python’s Dash framework and R’s shiny package so you will be able to observe, analyze and present various aspects of your business and other activities in effective ways. In addition I can build websites and mobile apps using React, Next.js and React Native.

I also have developed skills in DevOps / MLOps. In particular how to deploy Machine Leaning or Deep Learning model into production on a cloud based or any other web server. So this step is very important in data science paradigm as the value for model is generated during production environment.
